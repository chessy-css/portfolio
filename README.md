## 📁 项目一：带电粒子流在磁四极子透镜中的可视化

### 📌 项目简介
模拟带电粒子束流在磁四极子透镜中运动轨迹的3D可视化程序，直观展示非线性磁场对粒子聚焦的影响。

### 🎯 解决的问题
传统薄透镜近似公式在偏离理想条件时存在较大误差，抽象公式难以直观理解。本项目通过可视化手段，让这种非线性误差“看得见”。

### ✨ 技术亮点
- 独立完成从物理建模、数值求解到实时渲染的全流程
- 可视化呈现非傍轴条件下粒子轨迹偏移与焦距误差
- 直观对比近似公式的失效区域，并借此探究非线性效应的特点和薄透镜公式成立边界条件

### 🏆 成果
- 获校级电磁学小论文一等奖

## 📁 项目二：绿色县域拼车系统

### 🎬 演示视频
https://b23.tv/W2Bhy2h

### 📌 项目简介
针对县域农村“最后一公里”出行难题设计的多车协同智能拼车系统，以甘肃西和县为试点，实现高效分配与路径优化。

### 🎯 解决的问题
主流拼车平台在低密度县域覆盖不足、空驶率高、乘客叫车难。本项目通过专用算法，整合零散需求，降低绕路与碳排放。

### ✨ 技术亮点
- **多车协同分配算法**：几何启发式筛选 + 局部交换优化，平均绕路减少15%
- **智能路径规划**：集成TSP求解器（动态规划+MST近似），自动生成最优接人顺序
- **县域地名解析**：解决非标准地名识别盲区
- **轻量架构**：Python Flask + SQLite + 高德API，易于部署

### 🏆 成果
- 科学与社会研讨课 A+
- 分配成功率 >90%，日订单支持100+
